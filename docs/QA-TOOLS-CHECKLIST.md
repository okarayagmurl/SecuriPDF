## Kritik API alan adları

Form → Stirling proxy eşlemesi için sık kullanılan alanlar:

| Araç | Alan adı | Not |
|------|----------|-----|
| merge-pdfs | `fileInput` (multiple) | Sıra UI sürükle-bırak ile belirlenir |
| split-pages | `pageNumbers`, `splitType`, `splitValue`, `bookmarkLevel`, … | Moda göre dinamik endpoint |
| compress-pdf | `optimizeLevel` veya `expectedOutputSize`, `grayscale`, `linearize`, `lineArt` | Mod: kalite / hedef boyut |
| ocr-pdf | `languages[]`, `ocrType`, `ocrRenderType`, `deskew`, `clean`, … | Çoklu dil checkbox |
| add-password | `password`, `ownerPassword`, `keyLength`, `preventPrinting`, … | İzinler: `prevent*` checkbox |
| change-permissions | `ownerPassword`, `prevent*` | Mevcut sahip parolası gerekli |
| cert-sign | `certType`, `p12File` / `privateKeyFile`+`certFile` / `jksFile`, `password`, `showSignature`, `pageNumber`, `reason`, `location`, `name` | Sertifika türüne göre dosya alanları |
| compare | `fileInput1`, `fileInput2`, `highlightColor1`, `highlightColor2` | HTML çıktı |
| add-watermark | `watermarkType`, `watermarkText`, `watermarkStyle`, `fontSize`, `opacity`, `alphabet`, `customColor`, `watermarkImage`, `convertPDFToImage` | Metin/görsel mod |
| add-stamp | `stampType`, `stampText`, `stampImage`, `pageNumbers`, `position`, `fontSize`, `rotation`, `opacity`, `overrideX`, `overrideY` | Konum 1–9 ızgara |
| auto-redact | `redactPatternIds` (JSON), `customRedactRegex`, `redactSelection` (JSON), `redactColor`, `customPadding`, `convertPDFToImage` | Önizleme + tarama |
| rotate-pdf | `angle` | 90 / 180 / 270 |
| remove-pages / extract-pages | `pageNumbers` | Virgül veya aralık |
| rearrange-pages | `pageOrder` | Sürükle veya CSV |
| crop | `x`, `y`, `width`, `height` | Görsel kırpma alanı |
| add-page-numbers | `customMargin`, `fontSize`, `fontType`, `startingNumber`, `position`, `pageNumbers` | Konum 1–9 |
| scale-pages | `pageSize`, `scaleFactor` | KEEP / A4 / Letter vb. |
| add-image | `fileInput`, `imageFile`, `x`, `y`, `everyPage` | Koordinat pt |
| get-info-on-pdf / verify-pdf / validate-signature | `fileInput` | JSON rapor indirme |

## Notlar

- ZIP çıktısı üreten araçlar (split, extract-images vb.): indirilen dosya `.zip` olmalı.
- JSON rapor araçları: indirme `application/json` veya dosya indirme tetiklemeli.
- Deploy sonrası statik önbellek için `?v=` sürüm kontrolü — bkz. [TROUBLESHOOTING.md](TROUBLESHOOTING.md).
