      Kurulum tamamlama

      Installer sonrasi bu adimlari sirayla tamamlayin. Otomatik maddeler islem yaptikca yesile doner.

        Active Directory / LDAP

        Kaydedilen ayarlar vault volume'da saklanır. "Keycloak'a uygula" ile federation güncellenir.

        
      
        

      
        
        
        Vault varsayılanları
        
        
        
        
        
        
        Süre dolunca sabitlenmemiş belgeler otomatik arşive taşınır. Sabitlenen (pin) belgeler arşivlenmez.
        
        
        
        
        
        Depo yolları /vault-data altında oluşturulur. Liste modu girişte «Tümü» seçiliyken geçerlidir.
        
        
        
        
      
        

      
        
        
        E-posta (SMTP)
        
        
        Kullanicilarin arsiv belgelerini kendilerine gondermesi icin. Ayarlar vault volume'da saklanir; .env varsayilanlarini gecebilir.
        
        
        
        
        
        
        
        
        
        
        
          Ic ag relay (or. 192.168.x.x): Yok + port 25/587,
          kimlik dogrulama kapali (relay IP ile yetkilendirir).
          Docker sunucusunun IP'si relay'de izinli olmali.

        Araç erişim profilleri
        
        
        Profiller kullanıcılara atanır. Profilde tanımlı olmayan araçlar — lisans paketine sonradan eklense bile — o profildeki kullanıcılarda görünmez. Örnek: Muhasebe profilinde OCR yok; Tarama profilinde OCR var.
